# Http

## GithubProxy
- 版本：`>= v2.0.0-beta.37`
- 類型：`string`
- 建議：`https://mirror.ghproxy.com/`

對於可能在從 Github 下載資源時遇到困難的使用者（如在中國大陸），此選項允許他們為 github.com 設定代理，以提高可存取性。

## InsecureSkipVerify

- 版本：`>= v2.0.0-beta.37`
- 類型：`bool`

此選項用於設定 Nginx UI 伺服器在與其他伺服器建立 TLS 連接時是否跳過證書驗證。
