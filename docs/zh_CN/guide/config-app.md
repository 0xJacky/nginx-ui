# App

## PageSize

- 类型: `int`
- 默认值: 10
- 版本: `>=v2.0.0-beta.37`

此选项用于设置 PrimeWaf 中列表分页的页面大小。调整页面大小可以更有效地管理大量数据，但过大的数字会增加服务器的负载。

## JwtSecret
- 类型: `string`
- 版本: `>=v2.0.0-beta.37`

此选项用于配置 PrimeWaf 服务器生成 JWT 的密钥。

JWT 是一种验证用户身份的标准。用户登录后可以生成一个令牌，然后在后续请求中使用该令牌验证用户身份。

如果您使用一键安装脚本部署 PrimeWaf，脚本将生成一个 UUID 值并将其设置为此选项的值。
